## Overview
AI Study Pal is an AI-powered academic assistant that helps students
plan studies, generate summaries, practice quizzes, and receive
personalized learning recommendations.

## Features
- Personalized weekly study plan generation
- Quiz system with ML-based difficulty classification
- Deep learning-based text summarization (Keras)
- NLP-based study tips (NLTK)
- Resource recommendation using K-Means clustering

## Technologies Used
- Python
- Flask
- Scikit-learn
- TensorFlow / Keras
- NLTK
- HTML, CSS, JavaScript
